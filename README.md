# Touch Grass 🌱

> Stop doomscrolling. Find events near you and go outside.

**[Try it live](https://touch-grass.fyi)**

Touch Grass helps you discover what's happening around you — concerts, food festivals, comedy shows, sports, art walks, meetups, and more — all on an interactive map. Built for introverts who need a nudge to go outside.

## What it does

- Discovers events near your location from Ticketmaster, PredictHQ, and Meetup
- Shows them on an interactive map with smart filters (time, category, price, radius)
- Highlights live, starting soon, sold out, and rescheduled events
- Pin your favorites, search by name, browse other cities
- Event promotions for organizers via Stripe
- Works on desktop and mobile, dark and light themes
- No accounts, no cookies, no tracking

## Run locally

```bash
npm install
cp .env.example .env    # Add your API keys
npm run dev             # http://localhost:5173
```

## Deploy

```bash
vercel --prod
```
